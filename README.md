# Mexico Refugee System Analysis (2013-2018)

**Data analyst:** Gwendolyne Madrid Aman

## What this is
Public data analysis of Mexico's refugee/asylum system using:
- COMAR applications/resolutions (2013–Sept 2018)
- COMAR partial release via PNT (2022 and 2025, minors only)
- Grupos Beta humanitarian assistance (2013–2025)
- Foreign victims of crime (2013–2025)

## Key findings
- Pending cases: 4 (2016) → 12,417 (Sept 2018)
- 2018: "Pendiente" became the majority outcome category
- COMAR tracked LGBTQIA+ resolutions: 144 in 2017 → 36 by September 2018 → no public data since
- Grupos Beta atendidos in Baja California: +650% (2015–2023)
- Foreign victims reported: <10% annually
- 2026 partial release: excludes adults, excludes 2018–2021, excludes 2023–2024
